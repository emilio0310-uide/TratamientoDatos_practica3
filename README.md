# Titanic Data Analysis

## Descripción del dataset

Este proyecto analiza el dataset Titanic disponible en Kaggle, el cual contiene información sobre los pasajeros del Titanic y su supervivencia.

## Objetivo

Explorar los datos, identificar patrones y construir un modelo predictivo para determinar los factores que influyen en la supervivencia.

## Procesamiento de datos

Se realizaron los siguientes pasos:

- Eliminación de columnas irrelevantes
- Manejo de valores faltantes
- Transformación de variables categóricas

## Análisis exploratorio

Se analizaron variables como:

- Edad
- Género
- Clase del pasajero
- Puerto de embarque

## Visualización

Se generaron gráficos utilizando:

- matplotlib
- seaborn

## Machine Learning

Se implementó un modelo Random Forest utilizando scikit-learn para predecir la supervivencia.

## Resultados

El modelo obtuvo una precisión aproximada del 82%.

## Conclusiones

El género y la clase del pasajero fueron variables altamente relacionadas con la probabilidad de supervivencia.
